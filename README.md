# manager-gin
基于gin的管理系统

## 项目结构
```
├── config      配置文件
├── db          数据库文件
├── docs        文档
├── ui          前端项目
├── src         源代码目录
│   ├── app             应用目录
│   ├── common          公共模块
│   ├── framework       框架
│   ├── middleware      中间件
│   ├── global          全局变量
│   ├── routers         路由
│   ├── server          服务
│   ├── utils           工具
├── main.go    入口文件
├── go.mod
├── README.md
```
